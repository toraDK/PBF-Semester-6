# PBF Semester 6

Repositori ini berisi kumpulan hasil praktikum mata kuliah Pemrograman Berbasis Framework (PBF) Semester 6. Setiap topik praktikum disimpan per jobsheet agar terstruktur, dilengkapi dengan dokumentasi gambar, implementasi aplikasi (sebagian besar berbasis Next.js), serta laporan akhir di folder khusus.

## Deskripsi Proyek

Proyek ini disusun untuk:

1. Merekam progres pembelajaran framework web modern secara bertahap dari jobsheet ke jobsheet.
2. Mengelompokkan source code, aset dokumentasi, dan laporan agar mudah ditinjau ulang.
3. Menyediakan referensi terstruktur untuk evaluasi, presentasi, dan pengembangan lanjutan.

Secara umum, struktur repositori terdiri dari:

1. Folder jobsheet (`jobsheet2` sampai `jobsheet20`) yang berisi implementasi materi praktikum per pertemuan.
2. Folder `laporan_Praktikum` yang memuat laporan tertulis untuk setiap jobsheet.
3. Folder pendukung seperti `gambar_dokumentasi` untuk bukti hasil pengerjaan.
