Tugas 1<br>
Perbedaan Client Side Rendering, Server Side Rendering, Static Site Generation<br>
-> CSR adalah metode rendering di mana proses pengambilan data dan penampilan konten dilakukan di sisi browser user setelah halaman dimuat<br>
-> SSR adalah metode rendering di mana server memproses dan mengambil data terlebih dahulu, kemudian mengirimkan halaman HTML yang sudah lengkap ke browser user<br>
-> SSG adalah metode rendering di mana halaman sudah dibuat dan di-generate saat proses build aplikasi<br><br>
