F. Pertanyaan Evaluasi<br>
1. Apa perbedaan [id].js dan [...slug].js?
<br>-> jika [id] itu hanya bisa menangkap 1 parameter saja, sedangkan [...slug] bisa menangkap parameter tidak terbatas
2. Mengapa slug berbentuk array?
<br>-> karena slug menangkap lebih dari 1 parameter
3. Kapan sebaiknya menggunakan Link dan router.push()?
<br>-> Link digunakan saat melakukan perpindahan halaman tanpa ada logika tambahan sedangkan router.push() digunakan jika ada logika sebeum ke halaman yang dituju contohnya seperti form, login, register
4. Mengapa navigasi Next.js tidak me-refresh halaman?
<br>-> Karena Next.js menggunakan Client-Side Navigation
